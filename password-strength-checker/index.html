Password Strength

Password Rules:

Contains at Least 8 Characters
Contains at Least One Number(0-9) or More
Contains at Least One Uppercase Letter (A-Z) or More
Contains at Least One Lowercase Letter (a-z) or More
Contains at Least One Special Character (~ ! @ # $ % ^ & * ( ) _ + { } " ' : ? / \ | . ,) or More
No Space Allowed
